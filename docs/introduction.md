# Introduction

## Welcome to Onyx Protocol

Onyx Protocol is building the universal trust and reputation infrastructure for Web3.

Our mission is to make reputation portable, verifiable, and owned by the user rather than individual platforms.

Every blockchain user should have a persistent identity that represents their contributions, experience, achievements, and credibility across every ecosystem.

Instead of rebuilding trust on every application, users should carry their reputation wherever they go.

Onyx Protocol exists to make this possible.

---

## Why We Exist

Web3 has successfully decentralized ownership.

It has decentralized finance.

It has decentralized governance.

It has decentralized applications.

But trust remains fragmented.

Every protocol creates its own reputation system.

Every community measures contribution differently.

Every campaign rewards users independently.

There is no universal reputation.

There is no portable identity.

There is no common trust layer.

Onyx Protocol solves this challenge.
